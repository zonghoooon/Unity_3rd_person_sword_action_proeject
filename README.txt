This project is a project that implements catching monsters with a sword from the perspective of a 3rd person character.😎

W,A,S,D: Move Character 🧎‍♂️
Space: Jump 🐬
Left Shift: Run 🏃‍♂️
Left Mouse Click: Attack ⚔

You can defeat Goblins by attacking them.😈
And, goblins keep chasing you

Various attack skills and health bars will be implemented in the future.😧
